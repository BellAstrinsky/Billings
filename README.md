# Billings

Billings is a work assignment application from AcroCharge,
Written in node.js, react, TS.

## Installation

```
git clone https://github.com/BellAstrinsky/Billings.git
cd Billings/Client
npm install
```

## run
inside Client directory run:
```
npm start 
```
inside Server directory run:
```
node app.js
```
